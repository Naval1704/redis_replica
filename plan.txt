  redis.io/docs/latest/develop/reference/eviction/ (understand LRU strategy)   
  3. Redis Persistence (AOF section) —
  redis.io/docs/latest/operate/oss_and_stack/management/persistence/
  4. Winsock select() tutorial — look up "Winsock non-blocking server select" —
   MS docs on select(), ioctlsocket, closesocket  
  5. LRU Cache (LeetCode #146) — you've likely already solved this, same list +
   map pattern

  ---
  Windows Gotchas

  - #define FD_SETSIZE 1024 before #include <winsock2.h> (default is 64)       
  - #define WIN32_LEAN_AND_MEAN before #include <windows.h>
  - Use closesocket() not close()
  - Use ioctlsocket(fd, FIONBIO, &mode) not fcntl()
  - For fsync: _commit(_fileno(file_ptr)) or FlushFileBuffers()
